### 安装vue-cli

vue-cli可以按模板生成一个Vue工程<br>
安装：
```
npm install -g vue-cli
```

## 初始化

执行以下命令，并按提示输入相关信息
```
vue init xlxwhy/vue-template-mobile myproject
注：vue-cli不支持私服，只运行Github等线上平台
```
## 运行
执行以下命令
```
cd myproject
xsh npm i
npm run dev

```
